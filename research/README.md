# Research

Competitor analysis, user research, and visual reference reports.
**Read-only-ish** — these inform design decisions but aren't actively edited.

## Files

- **`watchtower-analysis.md`** — analysis of [Watchtower](https://watchtowerapp.com)
  (closest competitor in the Premiere folder-watching space). Behaviour,
  feature set, decision precedents we adopt or reject.
- **`watchtower-video/`** — frame-by-frame analysis of Watchtower's marketing
  video. `transcript.txt` + 55 numbered screenshots + `extract_frames.py`
  (the script that generated them).
- **`visual-reports/`** — ad-hoc visual diffs and progress snapshots from
  specific dates.
